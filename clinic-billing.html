门诊收费完整流程架构

从挂号到收费 — 全链路架构分层设计

标准流程

  • • 患者到院 → 挂号窗口建卡/挂号
  • • 挂号窗口 → 挂号服务 → 写入患者库
  • • 分诊候诊 → 医生工作站接诊
  • • 医生开处方/检查/检验
  • • 收费窗口 / 自助终端 → 收费服务

收费 & 结算流程

  • • 收费服务调用医保结算服务
  • • 医保结算 → 医保系统接口 (实时)
  • • 计算统筹支付 / 个人账户 / 自费
  • • 生成账单 → 银行/微信/支付宝支付
  • • 支付成功 → 写入收费记录库 → 打印发票

退费流程（虚线路径）

  • • 收费窗口发起退费申请
  • • 退费服务审核 (权限校验)
  • • 医保冲账 → 医保系统接口
  • • 支付原路退回 (银行/微信/支付宝)
  • • 写入冲账记录 → 收费记录库
